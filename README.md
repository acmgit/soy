# Soy


This Mod adds Soyplants with Beans to Minetest.
It supports farming and farming_redo from TenPlus.

![Screenshot 1](textures/soy_screenshot.jpg)

## Install

Move your Download to the Mods-Folder.

## Depends

default<br>
farming<br>
vessels<br>

### optional
mobs<br>
farming_redo<br>
